Features

This relay aggregates and validates high-quality Nostr user profiles with the following requirements:

  • Must have a name and bio
  • Profile picture must be at least 300x600 pixels
  • User must have published at least one text note
  • Excludes bridge accounts (mostr, ActivityPub)

API Endpoints

/random - Get random validated profiles

  • Query parameter: ?count=N (default: 12, max: 500)
  • Returns JSON by default
  • Returns HTML gallery if Accept header contains "text/html"
